# Bhavatarini Thangaraju — Portfolio

Personal portfolio website for Bhavatarini Thangaraju, Full-Stack Software Engineer.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 |
| Build Tool | Vite 5 |
| Animation | Framer Motion 11 |
| Styling | Inline styles + Tailwind CSS |
| Canvas | HTML5 Canvas API (ocean/wave background) |
| Fonts | Playfair Display, Inter (Google Fonts) |
| Deployment | Static site (Vite build) |

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
